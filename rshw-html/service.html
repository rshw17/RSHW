Hardware informações

Processador

O processador é o cérebro do computador, responsável por executar instruções e processar dados. A velocidade de clock e o número de núcleos são fatores-chave para avaliar o desempenho do processador.

motherboard

A motherboard é um componente que fornece uma plataforma para a interconexão de diversos compenentes de harware. Essa placa de circuito impresso abriga slots e conectores para CPU, RAM, placas de expansão e periféricos.

placa Gráfica

A placa gráfica, também conhecida como GPU, é um componente essencial em computadores e dispositivos eletrônicos, responsável pelo processamento de gráficos e imagens.

Armazenamento

O armazenamento funciona armazenando dados digitalmente em meios físicos, como discos magnéticos giratórios (HDs) ou chips de memória flash (SSDs), permitindo o acesso e a recuperação desses dados quando necessário.

Memórias Ram

A memória RAM é um tipo de memória volátil utilizada pelo computador para armazenar temporariamente dados e instruções que estão em uso ativo. Ela permite o acesso rápido aos dados pelo processador.

Bios

A BIOS é um firmware embutido na motherboard do computador, responsável por iniciar o hardware, realizar testes de diagnóstico durante a inicialização e fornecer uma interface para configuração básica do sistema antes da inicialização do sistema operativo.

placas de som

A placa de som converte sinais digitais em sinais analógicos para produzir áudio audível pelas colunas ou fones de ouvido. Ela processa e amplifica os dados de áudio recebidos pelo computador, permitindo a reprodução de música, sons de sistema e comunicação por voz.

Mais informações aqui
placa de ethernet

A placa de Ethernet é responsável por conectar como um computador, à rede local ou à Internet via cabo Ethernet. Ela converte os dados digitais em sinais elétricos que são transmitidos através do cabo para se comunicar com outros computadores na rede.

Placa de Expansão USB

Uma placa de expansão de USB funciona conectando-se a um slot de expansão para adicionar portas USB adicionais para conectar dispositivos externos. Ela amplia a capacidade de conexão USB do computador, permitindo mais dispositivos sejam conectados simultaneamente.

Mais informações aqui
Placa de Captura de Som

Uma Placa de Captura de som captura sinais de áudio de fontes externas, como microfones ou equipamentos de som, e os converte em dados digitais que podem ser processados ou gravados pelo computador. Estas placas permitem a gravação de áudio de alta qualidade diretamente no computador para produção de música, transmissões ao vivo, podcasts e muito mais.

Fonte de alimentação

A fonte de alimentação converte a corrente alternada (AC) da tomada em corrente contínua (DC) de baixa tensão, que é necessária para alimentar os componentes internos do computador. Ela distribui energia adequada a cada componente, garantindo um funcionamento estável e seguro do sistema.

Mais informações aqui
sistema de refrigeração

O sistema de refrigeração de um computador dissipa o calor gerado pelos componentes internos, como a CPU e a GPU, usando ventiladores, dissipadores de calor ou sistemas de resfriamento líquido. Isso mantém as temperaturas operacionais dentro de limites seguros, garantindo um desempenho estável e prevenindo danos por superaquecimento.

Periféricos

Os periféricos são dispositivos externos conectados a um computador que permitem entrada, saída e armazenamento de dados, como teclados, ratos, monitores e impressoras. Eles se comunicam com o sistema através de interfaces como USB, HDMI, ou Bluetooth, ampliando as funcionalidades do computador.

Caixa ou gabinete

A caixa ou gabinete do computador abriga e protege os componentes internos, como a motherboard, HD`s e a fonte de alimentação. Ele também facilita a ventilação e a gestão de cabos, garantindo uma organização eficiente e um fluxo de ar adequado para o arrefecimento.
